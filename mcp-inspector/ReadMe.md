## Connection to ezPresto MCP Server
with AIE 1.12 the ezPrestoMCP server is included as feature. In order to connect to it, provide the JWT token custom headers as header Authorization: Bearer "YOUR TOKEN" and the URL, something like "https://ezprestomcp.YOURAIEDOMAIN/mcp", you can retrieve from Data Sources in the URL section. Select Streamable HTTP as Type from the Dropdown.
